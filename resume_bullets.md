# Resume Bullets

## English

- Built **Jarvis**, an iOS personal AI assistant prototype using React Native, Expo, FastAPI, SQLite, and LLM orchestration.
- Designed bounded contexts for calendar, tasks, local services, document memory, travel, attachments, weather, and money/cost tracking.
- Implemented confirmation-first action safety for calendar mutations, compound schedule drafts, itinerary saving, and cost creation.
- Built simulator-based golden-journey QA covering weather brief, schedule changes, file preview, local-service planning, travel comparison, calendar verification, and cost tracking.
- Created a tiered regression workflow with fast 20/80 smoke checks, targeted scenario tests, UX replay review, and full regression gates before handoff.

## Chinese

- 构建 **Jarvis** iOS 个人 AI 助理原型，技术栈包括 React Native、Expo、FastAPI、SQLite 和 LLM orchestration。
- 按 bounded context 拆分日程、任务、本地服务、文档记忆、旅行、附件、天气和费用记录等模块。
- 实现确认优先的安全执行机制，用于日历修改、复合日程草稿、旅行行程保存和费用创建。
- 建立基于 iOS 模拟器的 golden journey QA，覆盖天气简报、日程改动、文件预览、餐厅推荐、旅行比较、日历验证和费用视图。
- 设计分层回归流程：快速 20/80 smoke test、场景化测试、UX replay review，以及交付前 full regression gate。
